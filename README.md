# Vortex Chess Engine

Vortex Chess Engine is a TypeScript-based chess engine designed for experimentation and learning. It features a modular architecture and is intended to be easy to extend and integrate.

## Features

- Written in TypeScript
- Modular core and evaluation logic
- UCI (Universal Chess Interface) handler
- Search and evaluation modules

## Getting Started

1. Install dependencies:
 ```bash
 npm install
 ```

2. Build the project:
 ```bash
 npm run build
 ```

3. Run the engine:
 ```bash
 npm start
 ```

## Project Structure

- `src/` - Main source code
  - `core/` - Core engine logic (e.g., UciHandler)
  - `eval/` - Evaluation functions
  - `search/` - Search algorithms
- `tests/` - Test files
